\# Développement de prototypes de jumeaux numériques géospatiaux et étude d'une architecture Digital Twin basée sur OpenTwins



\## Présentation



Ce dépôt GitHub regroupe l'ensemble des travaux réalisés dans le cadre de mon stage de Licence 3 Informatique, parcours \*\*Numérique pour les Environnements Connectés (NEC)\*\*, effectué au \*\*Laboratoire d'Informatique de l'Université de Pau et des Pays de l'Adour (LIUPPA)\*\*, situé au Pavillon Montaury à Anglet.



L'objectif principal du stage était de développer des prototypes de jumeaux numériques (Digital Twins) géospatiaux permettant de visualiser des données réelles, de réaliser des simulations et d'étudier une architecture basée sur \*\*OpenTwins\*\*.

\# Description des projets



\## Pilot 1



Prototype de Digital Twin géospatial permettant de simuler un épisode de pluie sur une zone définie par l'utilisateur et d'étudier son impact sur les cours d'eau et le réseau routier.



Technologies :



\* PostgreSQL

\* PostGIS

\* Node.js

\* Express

\* Leaflet



\---



\## Pilot 2



Prototype de Digital Twin géospatial consacré à l'étude de la répartition des espèces d'écrevisses en Roumanie. Il permet de visualiser les observations, les rivières et d'interroger le modèle numérique de terrain.



Technologies :



\* PostgreSQL

\* PostGIS

\* Node.js

\* Express

\* Leaflet



\---



\## OpenTwins



Cette partie regroupe les travaux réalisés autour de la plateforme OpenTwins :



\* découverte de Kubernetes ;

\* installation de K3s ;

\* installation de Helm ;

\* étude du démonstrateur Smart City ;

\* scripts Python testés sur Raspberry Pi.
