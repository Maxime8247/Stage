\# Pilot 2



Ce projet correspond au développement d'un prototype de Digital Twin géospatial consacré à l'étude des espèces d'écrevisses en Roumanie.



\## Structure du projet



\### api/



Contient le serveur Node.js permettant d'interroger les données géographiques stockées dans PostgreSQL/PostGIS.



\### carte/



Contient l'interface Leaflet permettant de visualiser les observations, les rivières et le modèle numérique de terrain.



\## Technologies



\- PostgreSQL

\- PostGIS

\- Node.js

\- Express

\- Leaflet
