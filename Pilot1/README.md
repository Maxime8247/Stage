\# Pilot 1



Ce projet correspond au développement d'un prototype de Digital Twin géospatial permettant de simuler des épisodes de pluie et leurs conséquences sur les cours d'eau et le réseau routier.



\## Structure du projet



\### api/



Contient le serveur Node.js développé avec Express permettant de communiquer avec la base de données PostgreSQL/PostGIS.



\### carte/



Contient l'interface web développée avec Leaflet permettant d'afficher les données géographiques et les simulations.



\## Technologies



\- PostgreSQL

\- PostGIS

\- Node.js

\- Express

\- Leaflet
