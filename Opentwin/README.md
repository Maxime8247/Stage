\# OpenTwins



Ce dossier regroupe les travaux réalisés autour de la plateforme \*\*OpenTwins\*\* dans le cadre du stage.



L'objectif était de découvrir le fonctionnement d'une architecture de \*\*Digital Twin\*\*, de préparer l'environnement de déploiement et d'étudier les différents composants utilisés par la plateforme.



\## Contenu du dossier



\### installation.md



Documentation des principales étapes d'installation :



\* kubectl

\* K3s

\* Helm

\* préparation de l'environnement OpenTwins



\### raspberry/



Scripts Python testés sur Raspberry Pi.



\* \*\*mqtt/\*\* : scripts de communication MQTT (tests fonctionnels réalisés avec un autre stagiaire).

\* \*\*hono/\*\* : premiers essais d'intégration avec Eclipse Hono (non finalisés).



\### smart-city/



Documentation concernant l'étude du démonstrateur \*\*OpenTwins Smart City\*\* et les essais réalisés pendant le stage.



\## Technologies utilisées



\* Kubernetes (K3s)

\* Helm

\* OpenTwins

\* Eclipse Ditto

\* Eclipse Hono

\* MQTT

\* Python



\## Remarque



Cette partie du dépôt a principalement pour objectif de présenter les différentes expérimentations réalisées durant le stage autour de l'écosystème OpenTwins et des technologies associées.
