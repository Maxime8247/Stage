\# Raspberry Pi



Ce dossier regroupe les différents scripts Python développés et testés sur la carte Raspberry Pi dans le cadre du stage.



\## Structure



\### mqtt/



Ce dossier contient les scripts utilisés pour communiquer via le protocole MQTT. Les tests ont été réalisés avec un autre stagiaire travaillant sur le même projet afin de vérifier le bon fonctionnement des échanges de messages entre les différents composants.



\*\*État :\*\* Fonctionnel.



\### hono/



Ce dossier contient les premiers essais réalisés avec Eclipse Hono dans le but d'étudier son intégration à l'architecture OpenTwins.



Bien que plusieurs tests aient été réalisés, cette partie n'a pas pu être entièrement finalisée durant le stage.



\*\*État :\*\* Prototype non finalisé.



\## Objectif



L'objectif de ces scripts était d'étudier les différents moyens de communication entre un Raspberry Pi et une plateforme de Digital Twin, ainsi que de comprendre le fonctionnement des protocoles utilisés dans l'écosystème OpenTwins.
